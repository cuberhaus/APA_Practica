# Security Policy — APA_Practica

## Reporting a Vulnerability
If you discover a security vulnerability, please email polcg10@gmail.com. Do not open a public issue.

## Security Considerations
- This is an academic ML project for hypothyroidism classification using Python and Jupyter notebooks.
- The medical dataset used (UCI Hypothyroid) is publicly available and contains no PII.
- All computation is local with no network or web components.
- Not intended for clinical or production use.
